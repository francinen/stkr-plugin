Stkr

Getting Started

  1. Download stkr.zip from GitHub
  2. Transfer stkr.js from the plugin folder to your project folder
  3. Link your html file to jQuery CDN and stkr.js by inserting these script tags immediately before your closing body tag:
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script><script src="sticky.js"></script>
    Make sure your src attribute includes the correct path to the stkr.js file.
  4. Give the desired sticky element an ID. 
    <img href="images/parachute.png" id="sticky">
    If you're including multiple fixed elements that will become visible at different points on your web page, they should each have a distinct IDs. 
    <img href="images/parachute.png" id="sticky1"><img href="images/cloud.png" id="sticky2"><img href="images/bird.png" id="sticky3">
  5. Call the stkr method in your JavaScript file. If you're calling the method on multiple elements, call them individually.
    $('#sticky1').stkr();$('#sticky2').stkr();$('#sticky3').stkr();

    By default, the sticky element remains fixed on the top-left corner for the entire length of your web page, with its top and left properties set to 20px.

    position: fixed,top: 20px,left: 20px,

Customizable Settings

  • Set custom start and end points

    • startSticky: Identify the page element that marks the point at which the sticky element becomes fixed to the page. If this is not set, the sticky element will automatically have a fixed position at the top of your web page. Value: String. If using an ID as the selector, include the hashtag (#idName) in the string. If using a class name, include the dot (.className) in the string.
    • endSticky: Identify the element that marks the point at which the selector becomes fixed to the page. If this is not set, the element will remain fixed for the remaining length of the document. Value: String. If using an ID as the selector, include the hashtag (#idName) in the string. If using a class name, include the dot (.className) in the string.
    • offsetStick: The sticky element becomes fixed a certain distance before or after its designated starting point. Value: Integer (Negative or Positive)
    • offsetUnstick: Offsets the designated end point by the given value. Value: Integer (Negative or Positive)
      • <section><img src="profile.jpg" id="sticky-profile"></section><section id="about-me">...</section><section>...</section><section id="portflio">...</section><footer>...</footer>
      <script>$('#sticky-profile').stkr({startSticky: '#about-me',endSticky: '#portfolio',offsetStick: 500,offsetUnstick: 200});</script>

  • Adjust position of fixed element

    • stickyPosition: Determines where the fixed element is positioned on the page. Values: 'top-left', 'top-right', 'bottom-left', 'bottom-right', or 'custom'
    • top, right, bottom, left: Control the element's horizontal or vertical offset from the edges of browser window. Value: Integer if using pixels (e.g., 30). String if using percentages (e.g., '30%').
    $('#sticky-profile').stkr({stickyPosition: 'custom',top: '50%',left: 40px});

  • Change orientation

    • Horizontal: Set to true if stkr method is triggered by a horizontal scroll. Value: Boolean (False by default)
